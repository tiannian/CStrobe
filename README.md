# CStrobe

> Strobe Protocol Framework in C++.

## Usage

Initialize `Strobe` statement.

``` c++
#include "cstrobe/strobe.h"

cstrobe::Strobe<uint64_t, 256> strobe; // Strobe-256/1600.
```
